BBBike Internals

New Layers

The following changes are necessary:

  • Create data/layer-orig file

  • Add entry to data/BASE

  • Add creation rule layer-orig => layer to data/Makefile. List the rule in TARGETS

  • Add check rules for layer to data/Makefile, if appropriate.

  • Add new layer to miscsrc/insert_points

  • Add new files to MANIFEST

  • Define a new abbreviation for layer ($abk) and use this abbreviation in the following bbbike definitions: %tag_group, %category_color ..., %str_file and/or %p_file, @normal_stack_order, array in set_bindings, enterpoint handling.
